Benchmark
------------------

Intersection between two sets A and B, M = |A| < N = |B| 

Complexities:
- STL O(M+N)
- Merge-based O(M+N) 
- Binary Search O(M*log(N))
- Mutual Partitioning O(M(1+log(N/M)))
- Doubling Search O(M(1+log(N/M)))
